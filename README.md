# data-cleaning-pandas
Primer proyecto Ironhack limpieza y análisis de dataset

En este proyecto se ha realizado la limpieza de un dataset dado para su posterior análisis con la libreria Pandas.

El supuesto escogido para su validación es el siguiente:

  - Se producen más ataques a hombres entre 20 y 40 años practicando actividades acuáticas.

Debido a que este supuesto no fue concluyente, se realizaron modificaciones para un mejor estudio del dataset.
